¿Qué es una variable?

Cuando hablamos de una variable en programación, nos referimos a una unidad utilizada para almacenar datos que se pueden modificar a partir de instrucciones en el código del programa.

Por lo tanto, es más fácil comprender una variable entendiéndola como un espacio de memoria que puede almacenar el nombre de un valor para guardar información de distintos tipos, el cual al adquirir un nombre puede ser diferenciado y llamado en distintas dependencias del algoritmo.

Antes de usar una variable se requiere la declaración de la misma. Para esto en JavaScript se requiere la siguiente estructura (Declaración): Nombre (El que recibe la variable), asignación (Signo de =) y valor (Lo que se va a almacenar dentro de la variable).

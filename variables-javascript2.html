Tipos de Variables.

• Entero: Números enteros.(X=1;) • Decimal: Números que contengan cifras decimales. Se debe usar un punto para indicar el decimal. (X=1.1)

Sintaxis

• Cadena de texto: Contenido alfanumérico, para usar este tipo de variables es obligatorio usar las comillas dobles. (X= “Hola”)

• Carácter: Aquí se puede almacenar símbolos cuya extensión sea exclusivamente un símbolo.

Sintaxis

-Declaramos una variable de nombre “María” y una asignamos la cadena “Hola a todos”.

Tipo

-JavaScript determina el tipo de variable a través del contenido, por lo cual la variable es de tipo cadena de caracteres (STRING).

<script>
var maria = "hola a todos"
</script>

Tipos de Variables.

• Entero: Como sugiere el nombre, este tipo de variables puede guardar los números enteros. Siendo así un número entero. (X=1;) • Decimal: Este tipo de variable nos permite almacenar números que contengan cifras decimales. Es decir números que no estén en su forma entera. Si desea almacenar números decimales, deberá usar un punto. (X=1.1;) Y en caso de querer añadir otro dato se usa la coma. (X=1,2;) • Cadena de texto: En este caso es cuando se desea almacenar contenido alfanumérico cuya extensión sea superior a un carácter, para usar este tipo de variables es obligatorio usar las comillas dobles. (X= “Hola”) • Carácter: Aquí se puede almacenar símbolos cuya extensión sea exclusivamente un símbolo.

Ver ejemplo

Paso 1: Dada la estructura esta declaración contiene un nombre que es “María” y una asignación que permite almacenar la cadena “Hola a todos”. Paso 2: JavaScript determina el tipo de variable a través del contenido almacenado, por lo cual la variable de nombre “María” es de tipo cadena de caracteres (STREAM). Paso 3: María= “Hola a todos”
